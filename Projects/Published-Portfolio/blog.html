My Journey as a CS Student

By Pratap Patra | B.Tech CSE | 5 min read

Introduction

I am a Computer Science Engineering student who believes in learning by building and staying consistent. In this blog, I share my learning journey, the challenges I faced, and how I am working towards becoming a skilled software developer.

How I Started

I began my programming journey with the C programming language. Learning C helped me understand core concepts such as memory management, logic building, and how programs work internally. Although challenging at first, it gave me a strong foundation.

What I'm Learning

  • C & C++ programming for fundamentals
  • Python for clean and readable code
  • Data Structures and Algorithms (arrays, linked lists, stacks, trees, graphs)
  • HTML, CSS & JS for Full Stack development

Challenges & Solutions

Throughout my learning process, I faced difficulties such as debugging errors, understanding logic, and managing time between college work and self-study. I handled these by practicing daily, reading errors carefully, and breaking complex problems into smaller steps.

Key Learnings

  • Consistency is more important than speed
  • Projects teach more than theory alone
  • Debugging improves problem-solving skills
